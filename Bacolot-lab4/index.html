Positioning of elements in CSS

Positioning the elements of an HTML page is no easy feat. CSS provides a set of methods that can be used cumulatively.

  • Margin and spacing: via the margin and padding properties
  • Centering: vertical or horizontal via the text-align and vertical-align properties
  • Display type: display property
  • Positioning: position property
  • Floating object: float property

1. Margin and Spacing

Margins and Spacing are used as a priority to position a block with respect to another block or with respect to its parent block. The margin property is used to apply outer spacing to the block and the padding property applies inner spacing to the block.

A paragraph of text with an inner spacing of 5 pixels all around the text (gray dotted border) wrapped in a parent block (blue border) and vertical outer spacing of 15 pixels and horizontal outer spacing of 10 pixels. 


<div style="border:2px solid blue;">
  <p style="padding:5px; margin:15px 10px; border:1px dashed gray;">
    A paragraph of text with an inner spacing of 5 pixels all around the text (gray dotted border)
    wrapped in a parent block (blue border) and vertical outer spacing of 15 pixels and horizontal outer spacing of 10 pixels.
  </p>
</div>

2. Horizontal and Vertical Centering

2.1 Horizontal Centering

To center text horizontally, the text-align property will be used. To center a block horizontally, one method is to use the automatic margins technique margin: auto - cf.above. Another is to make use of flexible with the justify-content property.

A paragraph of text with a width of 50% that of the parent block, centered horizontally, and whose text is also centered horizontally. 


<div style="border:2px solid blue;">
  <p style="text-align:center; width:50%; margin:auto; border:1px dashed gray;">
    A paragraph of text with a width of 50% that of the parent block, centered horizontally,
    and whose text is also centered horizontally.
  </p>
</div>

2.2 Vertical Centering

Vertical centering depends on the container element. If the container block has the display: table-cell property, you can use vertical-align: middle. Otherwise, you can use flexible boxes and the align-items property.

Text paragraph 50% wide, centered vertically and horizontally in its parent block. 


<div style="border:2px solid blue; height:250px; width:50%; display:flex; align-items:center; justify-content:center;">
  <p style="text-align:center; width:50%; border:1px dashed gray;">
    Text paragraph 50% wide, centered vertically and horizontally in its parent block.
  </p>
</div>

3. Display Type

Changing the rendering mode of HTML elements via the display property enables easy positioning of the elements relative to one another. Display Block/Inline - Inline-block - Table cell = Flex (spacebetween)

3.1 Display Block / Inline

By default, HTML elements use either block (e.g., <div>, <p>) or inline (e.g., <a>, <span>) rendering. Itâ€™s possible to modify this using CSS. For example, use a list of links to make a horizontal menu:

  • Home
  • Section 1
  • Section 2
  • Section 3
  • Contact

<nav>
  <ul class="menu">
    <li><a href="#">Home</a></li>
    <li><a href="#">Section 1</a></li>
    <li><a href="#">Section 2</a></li>
    <li><a href="#">Section 3</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

3.2 Display Inline-Block / Table-cell

The inline-block display lets you place elements side by side but keep block-like properties. Table-cell can also help vertically align text within containers. 


        <div>
    <label style="display: inline-block; width: 200px; color: green;">
        The inline-block label in my field
    </label>
    <input type="email" name="email" placeholder="Your email"/>
</div>

3.3 Display Flex

The flex display property provides a modern way to position items evenly. Using justify-content and align-items gives easy alignment. 


        <div style="display:flex; justify-content: space-between; border: 2px solid blue; height: 80px;">
    <div style="width:30%; margin:1.5%; border: 1px solid green">c</div>
    <div style="width:30%; margin:1.5%; border: 1px solid pink"></div>
    <div style="width:30%; margin:1.5%; border: 1px solid gray"></div>
</div>

4. Positioning Type

The position property lets you precisely place elements on a page. Options include fixed, relative, and absolute positioning.

4.1 Fixed Position

A fixed element stays in place when you scroll the page, like the header of this document. 


          header.header-fix {
            height: 85px;
            width: 100%;
            position: fixed;
            top: 0;
            left: 0;
            background-color: #eee;
            border-bottom: 1px solid #ddd;
            box-shadow: 0px 3px 5px rgba(0,0,0,0.75);
          }

4.2 Relative Position

An element with position: relative is moved relative to its normal position.

This block is offset by 10 pixels down and to the right. 

        <div style="display:flex; justify-content: space-between; border: 2px solid blue; height: 80px;">
    <div style="width:30%; margin:1.5%; border: 1px solid green">c</div>
    <div style="width:30%; margin:1.5%; border: 1px solid pink"></div>
    <div style="width:30%; margin:1.5%; border: 1px solid gray"></div>
</div>

4.3 Absolute Position

An element with position: absolute is positioned relative to its nearest positioned ancestor.

This block is offset by 10 pixels down and to the right with respect to the parent block. 

 
<div style="display:flex; justify-content: space-between; border: 2px solid blue; height: 80px; position: relative;">
    <div style="width:30%;margin:1.5%;border: 1px solid green"></div>
    <div style="width:30%;margin:1.5%;border: 1px solid pink;position: relative; top: 10px; left:10px;">
        This block is offset by 10 pixels down and to the right with respect to the parent block
    </div>
    <div style="width:30%;margin:1.5%;border: 1px solid gray"></div>
</div>

5. Floating Objects

The float property allows elements (like images) to align beside text. It was widely used for layouts before flex and grid.
